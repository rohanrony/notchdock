# Notchlet Monetization Roadmap

## Strategy
Use free core modules to build trust, then sell small premium extensions for $1 each.

## Free modules
- Calendar
- Clipboard
- Timer
- Music
- Claude basic chat

## Paid extensions
- Claude chat history
- Clipboard pinboard
- Pomodoro plus
- Future productivity extensions

## Tip jar
Offer $5, $10, and $20 support options.

## Rules
- Music stays free.
- No AI clipboard actions.
- **Compliance**: Use StoreKit 2 for all transactions. No external payment links (e.g., Stripe/PayPal) to comply with App Store guidelines.
- Premium features should feel optional, not blocked.
